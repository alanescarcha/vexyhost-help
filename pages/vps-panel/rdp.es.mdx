# Conexión mediante RDP (Remote Desktop Protocol)

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Callout>
RDP (Remote Desktop Protocol) sólo está disponible en Windows, este protocolo es exclusivo de Windows Server.
</Callout>

<Steps>
### Paso 1
Abra el cliente RDP en su PC, puede buscar `RDP` en el buscador de Windows.

### Paso 2
Cuando abras el programa, debes ingresar la IP de tu VPS y luego hacer click en `Connect`.
![](https://i.imgur.com/gTvfFN2.png)

<Callout type="info">
El usuario normalmente es **Administrator**, en caso contrario se le enviará el usuario y contraseña a su correo electrónico. Si ves un mensaje de credenciales, simplemente haz clic en continuar.
</Callout>
</Steps>
