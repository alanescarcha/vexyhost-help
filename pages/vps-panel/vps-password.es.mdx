# Cambiar contraseña del VPS

import { Callout } from 'nextra/components'

<Callout>
Si olvidaste la contraseña de tu VPS, ya sea Windows o Linux esta guía te puede ayudar. **Recuerda usar contraseñas seguras.**
</Callout>

## Linux
Vaya a su [panel VPS](https://vps.vexyhost.com/). Ve a `Settings` y luego en la pestaña `Change Password`.
![](https://i.imgur.com/FpFb6Li.png)

<Callout type="info">
Puede que necesites **reiniciar el VPS desde el panel** para que se establezca la nueva contraseña. Para saber cómo reiniciar tu VPS desde el panel sigue [este artículo](/vps-panel/start-vps).
</Callout>

## Windows
Vaya a su [panel VPS](https://vps.vexyhost.com/). Ve a `Settings` y luego en la pestaña `Change Password`.
<Callout type="warning">
Deje el campo `Windows Admin Name` en blanco para poder cambiar la contraseña del Administrador.
</Callout>
![](https://i.imgur.com/7QKEgpp.png)

<Callout type="info">
Puede que necesites **reiniciar el VPS desde el panel** para que se establezca la nueva contraseña. Para aprender a reiniciar tu VPS desde el panel sigue [este artículo](/vps-panel/start-vps).
</Callout>
