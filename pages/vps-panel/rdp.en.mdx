# Connect via RDP (Remote Desktop Protocol)

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Callout>
RDP (Remote Desktop Protocol) is only available on Windows, this protocol is unique to Windows Server.
</Callout>

<Steps>
### Step 1
Open the RDP client on your PC, you can search for `RDP` in Windows.

### Step 2
When you open the program, you must enter the IP of your VPS and then click on `Connect`.
![](https://i.imgur.com/N9ZkZQt.png)

<Callout type="info">
The user is normally **Administrator**, otherwise the username and password will be sent to your email. If you get a credentials message, simply click continue.
</Callout>
</Steps>
