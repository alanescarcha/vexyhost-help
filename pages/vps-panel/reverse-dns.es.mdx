# Reverse DNS / PTR Record

import { Callout } from 'nextra/components'

<Callout>
Los servidores de correo electrónico utilizan registros PTR para verificar la autenticidad de las direcciones IP de envío. Los registros PTR pueden ayudar a evitar que los correos electrónicos sean etiquetados como spam o rechazados por otros servidores de correo como Gmail. **Recuerda que también debes mantener una buena reputación de la IP del VPS, si envías SPAM, nada podrá ayudarte, además podemos llegar a suspender tu VPS por este motivo.**
</Callout>

Por el momento no es posible cambiar los DNS inversos y el registro PTR de las IPs de tu VPS desde el panel del VPS. Debes solicitar el cambio mediante ticket de soporte, este es totalmente gratuito. **Recuerda especificar la IP a cambiar y el nuevo hostname o dominio.**
