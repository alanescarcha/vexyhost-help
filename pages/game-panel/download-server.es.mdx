# Descarga tu servidor

Si necesitas descargar todos los archivos de tu servidor, puedes generar una copia de seguridad y descargarla. Para saber cómo generar una copia de seguridad consulta [este artículo](/game-panel/backups).

![](https://i.imgur.com/xdcOkq8.png)

import { Callout } from 'nextra/components'

<Callout type="warning" emoji="⚠️">
    **Si tienes bases de datos, éstas no formarán parte de la copia de seguridad**. Para descargar tus bases de datos, debes hacerlo desde phpMyAdmin exportando cada una de ellas. Si no sabes cómo hacerlo consulta [este artículo](/game-panel/databases).
</Callout>
