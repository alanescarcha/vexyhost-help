# Download your server

If you need to download all the files from your server, you can generate a backup and download it. To learn how to generate a backup check [this article](/game-panel/backups).

![](https://i.imgur.com/xdcOkq8.png)

import { Callout } from 'nextra/components'

<Callout type="warning" emoji="⚠️">
    **If you have databases, these will not be part of the backup**. To download your databases, you must do it from phpMyAdmin exporting each one. If you don't know how to do that check [this article](/game-panel/databases).
</Callout>
