# Configurar un subdominio

Nuestro panel de juego le permite configurar un subdominio personalizado para su servidor. Para crear un subdominio, vaya a la sección Subdomains, escriba el prefijo del subdominio y luego el dominio que desea utilizar. Tenemos algunas opciones en el menú desplegable para que elijas.

![](https://i.imgur.com/Nu3SaDi.png)

A continuación, puedes utilizar el subdominio seleccionado para conectarte a tu servidor. Ten en cuenta que algunos juegos no soportan subdominios o registros SRV, para aquellos juegos que soportan subdominios y dominios pero no un registro SRV, necesitarás adquirir una IP dedicada.
