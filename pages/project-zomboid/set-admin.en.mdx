# How to be an Admin on your Server in Project Zomboid

import { Callout, Steps, Code } from 'nextra/components'

<Callout type="warning">
    In this process, you will become an Admin on your server. Please do not do this on players you do not trust as they will have admin level access on your Project Zomboid Server
</Callout>

<Steps>

### Step 1

Go to Console in the [Game Panel](https://panel.vexyhost.com)

### Step 2

Type the following command to add yourself as an Admin: <Code>setaccesslevel \<your user\> admin</Code>

Current levels: Admin, Moderator, Overseer, GM, Observer

![](https://i.imgur.com/5rn6lIT.png)

### Step 3

When you execute the command, the server will respond something like this:

```log
LOG  : General     , 1739387602958> 6,822,289,647> command entered via server console (System.in): "setaccesslevel Alan admin"
LOG  : General     , 1739387602999> 6,822,289,689> User Alan is now admin
```

### Step 4

Now when you enter the game you will see the admi menu in-game

![](https://i.imgur.com/SlVQMym.png)

</Steps>
