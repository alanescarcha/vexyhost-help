# Cómo ser administrador de tu servidor en Project Zomboid

import { Callout, Steps, Code } from 'nextra/components'

<Callout type="warning">
    En este proceso, te convertirás en administrador de tu servidor. Por favor, no hagas esto con jugadores en los que no confíes, ya que tendrán acceso a nivel de administrador en tu servidor del Proyecto Zomboid.
</Callout>

<Steps>

### Paso 1

Vaya a Consola en el [Game Panel](https://panel.vexyhost.com)

### Paso 2

Escriba el siguiente comando para añadirse como administrador: <Code>setaccesslevel \<tu usuario\> admin</Code>

Niveles disponibles: Admin, Moderator, Overseer, GM, Observer

![](https://i.imgur.com/5rn6lIT.png)

### Paso 3

Cuando ejecute el comando, el servidor responderá algo parecido a esto:

```log
LOG  : General     , 1739387602958> 6,822,289,647> command entered via server console (System.in): "setaccesslevel Alan admin"
LOG  : General     , 1739387602999> 6,822,289,689> User Alan is now admin
```

### Paso 4

Ahora cuando entres en el juego verás el menú admin en el juego

![](https://i.imgur.com/SlVQMym.png)

</Steps>

✅ Eso es todo. Si tienes algún problema, ponte en contacto con nosotros por [Discord](https://discord.vexyhost.com/) para que podamos ayudarte.
