# Cómo instalar Python

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Steps>

### Paso 1
Ve a la pestaña **Discord Bot Installer** en tu Game Panel.

### Paso 2
Instala la versión **Python generic**.

![](https://i.imgur.com/yXMshmm.png)

### Paso 3
Una vez completada la instalación, tu servidor se reinstalará automáticamente, solo espera a que termine.

![](https://i.imgur.com/BifZIn7.png)

### Paso 4
Por último, recuerda elegir en la pestaña Startup la versión de Python que quieres ejecutar en el servidor, **la versión de Python que elijas debe ser compatible con tu bot.**

![](https://i.imgur.com/EaB3WkQ.png)

### Paso 5
Sube los archivos de tu bot e inicia el servidor. **Recuerda subir tu requirements.txt para que el servidor sepa qué paquetes y dependencias instalar para tu bot.**

<Callout type="info">
Por defecto el archivo que intentará iniciar el servidor es app.py puedes cambiarlo desde la pestaña Startup.
</Callout>

</Steps>

✔ ¡Listo! Una vez que subas tus archivos podrás iniciar el servidor, si tienes algún inconveniente puedes contactarnos por [Discord](https://discord.vexyhost.com/).
