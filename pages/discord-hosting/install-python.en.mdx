# How to install Python

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Steps>

### Step 1
Go to **Discord Bot Installer** tab in your Game Panel.

### Step 2
Install the **Python generic** version.

![](https://i.imgur.com/yXMshmm.png)

### Step 3
Once the installation is complete, your server will automatically reinstall, just wait for it to finish.

![](https://i.imgur.com/BifZIn7.png)

### Step 4
Finally, remember to choose from the Startup tab the version of Python you want to run on the server, **the version of Python you choose must be compatible with your bot.**

![](https://i.imgur.com/EaB3WkQ.png)

### Step 5
Upload your bot files and start the server. **Remember to upload your requirements.txt so the server knows what packages and dependencies to install for your bot.**

<Callout type="info">
By default the file that will try to start the server is app.py you can change this from the Startup tab.
</Callout>

</Steps>
