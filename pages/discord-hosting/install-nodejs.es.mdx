# Cómo instalar NodeJS

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Steps>

### Paso 1
Ve a la pestaña **Discord Bot Installer** en tu Game Panel.

### Paso 2
Instala la versión **Node.js generic**.

![](https://i.imgur.com/hFLc2y7.png)

### Paso 3
Una vez completada la instalación, tu servidor se reinstalará automáticamente, solo espera a que termine.

![](https://i.imgur.com/BifZIn7.png)

### Paso 4
Por último, recuerda elegir en la pestaña Startup la versión de NodeJS que quieres ejecutar en el servidor, **la versión de NodeJS que elijas debe ser compatible con tu bot.**

![](https://i.imgur.com/HBEeVQ8.png)

### Paso 5
Sube los archivos de tu bot e inicia el servidor. Si tu bot necesita configuraciones como variables de entorno puedes crear el archivo .env y colocar allí tus variables de entorno.

<Callout type="info">
Por defecto se descarga un sencillo código de ejemplo desde el repositorio de GitHub https://github.com/vexyhost/discord-js-bot puedes eliminar esos archivos para subir tu bot.
Por defecto el archivo que intentará iniciar el servidor es index.js puedes cambiarlo desde la pestaña Startup.
</Callout>

</Steps>

✔ ¡Listo! Una vez que subas tus archivos podrás iniciar el servidor, si tienes algún inconveniente puedes contactarnos por [Discord](https://discord.vexyhost.com/).
