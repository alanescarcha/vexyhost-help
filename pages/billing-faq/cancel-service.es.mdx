# ❌ ¿Cómo cancelar un servicio?

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Steps>

### Paso 1
Inicie sesión en [su área de clientes](https://billing.vexyhost.com/clientarea.php). Busca el servicio activo que quieres cancelar.

### Paso 2
En la sección de acciones busque la opción `Solicitar Cancelación`.

![](https://i.imgur.com/jubDbM0.png)

### Paso 3
A continuación debes escribir el motivo de la cancelación y el tipo de cancelación.

<Callout type="info">
**Tipos de cancelación:**
- **Immediata:** El servicio se cancelará y finalizará a las 00:00hs UTC del mismo día de la solicitud de cancelación.

- **Final del periodo del contrato:** El servicio se cancelará y finalizará el último día del contrato a las 00:00hs UTC.
</Callout>

![](https://i.imgur.com/hxzYMEr.png)

</Steps>

✔ ¡Listo! Una vez envíes la solicitud de baja tu servicio se dará de baja automáticamente según el tipo de baja elegido. **En caso de que te arrepientas de la cancelación puedes contactar a soporte vía ticket para cancelar la solicitud de cancelación.**
