# Cómo cambiar la configuración del servidor Palworld

import { Callout, Steps, Code } from 'nextra/components'

<Callout type="warning">
    Este juego está en Early Access, por lo que cambiar la configuración puede resultar confuso al principio. Algunos parámetros como **ServerPlayerMaxNum**, **ServerName**, **ServerPassword** y **AdminPassword** se configuran desde la pestaña Startup.
</Callout>

<Callout type="info">
## Problemas conocidos / Preguntas frecuentes
1. El servidor no aparecerá en la pestaña Community-Server.<br/>
-> Este es un problema conocido y los desarrolladores esperan solucionarlo lo antes posible. La mejor opción es conectarse por IP y contraseña.
2. El servidor tiene una fuga de memoria.<br/>
-> También son conscientes de ello.<br/>
-> La opción bEnableInvaderEnemy parece tener un gran impacto en el uso actual de RAM. Deshabilitarla podría ser una opción.<br/>
-> Consejo: Programe un reinicio del servidor cada 6h. Puede ajustar el valor en función de su memoria RAM.
3. El servidor no aparece en la lista de servidores de Steam.<br/>
-> Es posible que esto no sea compatible nunca.
4. El archivo de configuración se borra/restablece al reiniciar el servidor.<br/>
-> Si el puerto de su servidor es diferente al puerto por defecto (8211), debe ajustar el puerto en el archivo de configuración. <Code>Pal/Saved/Config/LinuxServer/PalWorldSettings.ini</Code>.<br/>
-> Antes de editar el archivo de configuración, asegúrese siempre de detener completamente el servidor. De lo contrario, no se guardarán todos los cambios.
</Callout>

<Steps>

### Paso 1

Asegúrese de que el servidor está desconectado.

![](https://i.imgur.com/gRntjqt.png)

### Paso 2

Localiza el archivo <Code>DefaultPalWorldSettings.ini</Code> y ábrelo haciendo clic sobre él.

![](https://i.imgur.com/sYL2uBp.png)

### Paso 3

Cuando abras el archivo <Code>DefaultPalWorldSettings.ini</Code> verás algo parecido a la siguiente imagen. Simplemente copia todo el contenido.

![](https://i.imgur.com/G7QRX6m.png)


### Paso 4

Ahora ve a la carpeta <Code>Pal > Saved > Config > LinuxServer</Code> y abre el archivo <Code>PalWorldSettings.ini</Code>. Dentro del archivo debe pegar el contenido copiado anteriormente desde <Code>DefaultPalWorldSettings.ini</Code>

![](https://i.imgur.com/PltTfx0.png)

![](https://i.imgur.com/x8wpOi8.png)

### Paso 5

Ahora puedes editar los parámetros y ajustes que necesites, **recuerda guardar el archivo usando el botón azul que dice "Save content"**.

<Callout type="info">
    Al iniciar y detener el servidor se borra el contenido del archivo PalWorldSettings.ini, recomendamos guardarlo en algún lugar de su PC para recordar los parámetros o ajustes realizados.
</Callout>


</Steps>

## Explicación de los parámetros

| Parámetro                                     | Descripción                                                                        |
|-----------------------------------------------|------------------------------------------------------------------------------------|
| Difficulty                                    | Ninguno - No parece importar para servidores dedicados.                            |
| DayTimeSpeedRate=1.000000                     | Cambia la duración del día.                                                        |
| NightTimeSpeedRate=1.000000                   | Cambia la duración de la noche. Sugerido 5.000000                                  |
| ExpRate=1.000000                              | Modifica cuanta experiencia obtienes.                                              |
| PalCaptureRate=1.000000                       | La probabilidad de capturar Pals.                                                  |
| PalSpawnNumRate=1.000000                      | La frecuencia con la que aparecen los Pals.                                        |
| PalDamageRateAttack=1.000000                  | Cuánto daño hacen los Pals.                                                        |
| PalDamageRateDefense=1.000000                 | Cuanto daño reciben los Pals.                                                      |
| PlayerDamageRateAttack=1.000000               | Cuánto daño hacen los jugadores.                                                   |
| PlayerDamageRateDefense=1.000000              | Cuánto daño reciben los jugadores.                                                 |
| PlayerStomachDecreaseRate=1.000000            | Ritmo al que disminuye el hambre.                                                  |
| PlayerStaminaDecreaseRate=1.000000            | Velocidad a la que disminuye la resistencia.                                       |
| PlayerAutoHPRegeneRate=1.000000               | La rapidez con la que se recuperan automáticamente los PS al recibir daño.         |
| PlayerAutoHpRegeneRateInSleep=1.000000        | Cuántos PS se regeneran mientras duermes.                                          |
| PalStomachDecreaseRate=1.000000               | Velocidad a la que disminuye el hambre de Pal.                                     |
| PalStaminaDecreaseRate=1.000000               | Velocidad a la que disminuye la resistencia de Pal.                                |
| PalAutoHPRegeneRate=1.000000                  | Ritmo con el que los PS de Pal se recuperan automaticamente al ser herido.         |
| PalAutoHpRegeneRateInSleep=1.000000           | Cuantos PS se regeneran mientras duermes.                                          |
| BuildObjectDamageRate=1.000000                | Daño a objetos.                                                                    |
| BuildObjectDeteriorationDamageRate=1.000000   | Cuanto daño recibirán los objetos construidos con el tiempo.                       |
| CollectionDropRate=1.000000                   | La velocidad a la que se recogen objetos de árboles o rocas.                       |
| CollectionObjectHpRate=1.000000               | Cuántos PS tienen los objetos rompibles del mundo.                                 |
| CollectionObjectRespawnSpeedRate=1.000000     | Ritmo con el que reaparecen objetos como árboles.                                  |
| EnemyDropItemRate=1.000000                    | Frecuencia con la que los enemigos sueltan objetos.                                |
| DeathPenalty=All                              | Determina lo que se tira al morir. Puede establecerse en ninguno.                  |
| bEnableAimAssistPad=True                      | Permite activar o desactivar el asistente de puntería.                             |
| bEnableAimAssistKeyboard=False                | Permite activar o desactivar el asistente de puntería.                             |
| DropItemMaxNum=3000                           | Cantidad máxima de objetos soltados permitidos a la vez.                           |
| BaseCampMaxNum=128                            | Cantidad máxima de bases que se pueden construir a la vez.                         |
| BaseCampWorkerMaxNum=15                       | Cantidad máxima de trabajadores que pueden estar en un campamento.                 |
| DropItemAliveMaxHours=1.000000                | Cuánto tiempo permanecerán los objetos soltados antes de desaparecer.              |
| bAutoResetGuildNoOnlinePlayers=False          | Si es verdadero, un clan se disolverá si no hay jugadores conectados.              |
| AutoResetGuildTimeNoOnlinePlayers=72.000000   | La rapidez con la que se disolverá un clan inactivo.                               |
| GuildPlayerMaxNum=20                          | Número máximo de jugadores del clan.                                               |
| PalEggDefaultHatchingTime=72.000000           | Cuánto tardan en eclosionar los huevos de Pal. Sugerido 15.000000                  |
| WorkSpeedRate=1.000000                        | Cómo funcionan rápidamente los Pals.                                               |
| CoopPlayerMaxNum=4                            | Número máximo de jugadores en un grupo.                                            |
| ServerPlayerMaxNum=32                         | Máximo de jugadores permitidos en un servidor (límite de 32).                      |
| ServerName="Default Palworld Server"          | Nombre del servidor                                                                |
| ServerDescription=""                          | Lo que se muestra al seleccionar el servidor en la lista. No tocar esto            |
| AdminPassword=""                              | Contraseña utilizada para conceder acceso de administrador.                        |
| ServerPassword=""                             | Contraseña necesaria para que se unan los jugadores normales.                      |

<Callout type="warning">
