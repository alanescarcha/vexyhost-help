# Cómo establecer una contraseña de administrador en un servidor Palworld

import { Callout, Steps, Code } from 'nextra/components'


<Steps>

### Paso 1

Asegúrese de que el servidor está apagado.

![](https://i.imgur.com/gRntjqt.png)

### Paso 2

Vaya a la pestaña Startup en su Game Panel y coloque una contraseña debajo de "ADMIN PASSWORD".

![](https://i.imgur.com/Hq6xBun.png)

### Paso 3

Inicia tu servidor y dentro del juego escribe en el chat <Code>/AdminPassword \<tu contraseña aqui></Code>. Por ejemplo: <Code>/AdminPassword contraseña1234</Code>

</Steps>

## Comandos de Admin

| Comando en el juego                    | Descripción                                                                                  |
|----------------------------------------|----------------------------------------------------------------------------------------------|
| /Shutdown \{Seconds} \{MessageText}    | El servidor se apaga después del número de Segundos. Será notificado de su MessageText.      |
| /DoExit                                | Forzar el apagado del servidor.                                                              |
| /Broadcast \{MessageText}              | Enviar mensaje a todos los jugadores del servidor.                                           |
| /KickPlayer \{SteamID}                 | Expulsar al jugador del servidor.                                                            |
| /BanPlayer \{SteamID}                  | Banear al jugador del servidor.                                                              |
| /TeleportToPlayer \{SteamID}           | Teletranspórtate a la ubicación actual del jugador objetivo.                                 |
| /TeleportToMe \{SteamID}               | El jugador objetivo se teletransporta a tu ubicación actual.                                 |
| /ShowPlayers                           | Mostrar información sobre todos los jugadores conectados.                                    |
| /Info                                  | Mostrar información del servidor.                                                            |
| /Save                                  | Guardar los datos del mundo.                                                                 |
