# How to set an Admin Password on a Palworld server

import { Callout, Steps, Code } from 'nextra/components'


<Steps>

### Step 1

Make sure that server is offline.

![](https://i.imgur.com/gRntjqt.png)

### Step 2

Go to the Startup tab and place a password below "ADMIN PASSWORD".

![](https://i.imgur.com/Hq6xBun.png)

### Step 3

Start your server and inside the game type in the chat <Code>/AdminPassword \<your password here></Code>. For example: <Code>/AdminPassword password1234</Code>

</Steps>

## Admin Commands

| In-game command                        | Description                                                                                  |
|----------------------------------------|----------------------------------------------------------------------------------------------|
| /Shutdown \{Seconds} \{MessageText}    | The server is shut down after the number of Seconds. Will be notified of your MessageText.   |
| /DoExit                                | Force stop the server.                                                                       |
| /Broadcast \{MessageText}              | Send message to all players in the server.                                                   |
| /KickPlayer \{SteamID}                 | Kick player from the server.                                                                 |
| /BanPlayer \{SteamID}                  | BAN player from the server.                                                                  |
| /TeleportToPlayer \{SteamID}           | Teleport to the current location of the target player.                                       |
| /TeleportToMe \{SteamID}               | Target player teleports to your current location.                                            |
| /ShowPlayers                           | Show information on all connected players.                                                   |
| /Info                                  | Show server information.                                                                     |
| /Save                                  | Save the world data.                                                                         |
