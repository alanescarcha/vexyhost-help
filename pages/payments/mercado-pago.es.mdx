# 🤝 Mercado Pago 🇦🇷

import { Callout } from 'nextra/components'
import { Steps } from 'nextra/components'

<Callout>
La activación del servicio con este medio de pago es inmediata. **Este método de pago sólo está disponible para Argentina 🇦🇷.**
</Callout>

<Steps>

### Paso 1 - Buscar la factura
Vaya a [sus facturas](https://billing.vexyhost.com/clientarea.php?action=invoices). Busque la factura que desea pagar y haga clic en ella.

### Paso 2 - Seleccioná la forma de pago.
Allí deberás elegir la opción **Mercado Pago - ARS (Solo Argentina - Only Argentina)** y luego hacer click en el botón "Pagar Ahora".

![](https://i.imgur.com/fIn2PPK.png)

A continuación será redirigido a Mercado Pago. Deberás elegir si pagar con el saldo de tu cuenta de Mercado Pago, con tarjeta de débito o crédito, o pagando en efectivo en Pago Fácil o Rapipago.

<Callout type="info">
No necesitás una cuenta de Mercado Pago para pagar con tarjeta de débito o crédito o en efectivo (Pago Fácil o Rapipago).
</Callout>

![](https://i.imgur.com/ZtTOEKe.png)

### Paso 3 - Pago
Por último, deberás hacer click en el botón "Pagar".

![](https://i.imgur.com/sTh6YlN.png)

</Steps>

✔ ¡Genial! Una vez que realices el pago, te enviaremos un correo electrónico confirmando el pago y la activación o renovación del servicio.
