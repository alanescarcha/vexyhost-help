# Cómo colocar el icono de tu servidor de Minecraft

import { Steps } from 'nextra/components';

<Steps>

### Paso 1

Primero necesitas tener la imagen que vas a usar como icono en tu servidor, esta imagen debe ser en **formato PNG** y debe tener un tamaño de **64x64 pixeles**, de lo contrario el icono no va a cargar en el servidor.

### Paso 2

Despues de tener nuestra imagen en formato .png y en tamaño 64x64 pixeles, debes renombrar el archivo a: `server-icon.png` y subirlo al servidor. Puedes subir el archivo `server-icon.png` arrastrandolo y subiendolo desde el File Manager en el Game Panel.

### Paso 3
Reinicia tu servidor y ahora deberias poder ver el icono de tu servidor en el juego.

![](https://i.imgur.com/DUdzXoD.png)

</Steps>
