# Cómo colocar el servidor en modo No Premium

import { Steps } from 'nextra/components';

<Steps>

### Paso 1

Detenemos nuestro servidor y en el archivo `server.properties` cambiamos la opción `online-mode` a `false`.

```properties filename="server.properties" copy=true
online-mode=false
```

### Paso 2
Inicia tu servidor y ahora los jugadores podrán unirse sin tener una cuenta premium o de pago.

![](https://i.imgur.com/DUdzXoD.png)
