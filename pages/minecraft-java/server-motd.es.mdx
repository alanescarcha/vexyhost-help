# Cómo colocar el MOTD a tu servidor de Minecraft

import { Steps, Code } from 'nextra/components';

<Steps>

### Paso 1

Nos dirigimos a algun sitio web que permita crear nuestra descripción del servidor de forma mas sensilla, recomendamos paginas como estas:

- https://motd.gg
- https://mctools.org/motd-creator

### Paso 2

En nuestro caso vamos a usar motd.gg, alli creamos nuestro motd. Tambien puedes copiar el motd de cualquier servidor usando el boton "Grab from server"

![](https://i.imgur.com/flVuGfU.png)

### Paso 3

Por ultimo cuando terminemos de crear nuestro motd usamos el boton "Get your MOTD" y lo copiamos en nuestro archivo <Code>server.properties</Code> de nuestro servidor. (Recuerda tener marcada la opción "Escape" en motd.gg)

![](https://i.imgur.com/5oEvgcI.png)

</Steps>

![](https://i.imgur.com/IQ82gOx.png)
