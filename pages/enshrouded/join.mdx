# How to join to your Enshrouded server

import { Callout, Steps, Code } from 'nextra/components'

<Callout type="info">
    This game has no way to join a server by IP from within the game, you must add the server to your Steam favorites list to be able to join. **Servers do not work on pirated copies of the game.**
</Callout>

<Steps>

### Step 1

Stop your server. Go to Network tab and click on **Create Allocation**.

![](https://i.imgur.com/H4Qp6W0.png)

### Step 2

Copy the new port. This will create a new port that you will use to join the server.

![](https://i.imgur.com/K1qYIfC.png)

### Step 3

Go to Startup tab and paste the new port under **SERVER QUERY PORT**.

![](https://i.imgur.com/8kxgZvQ.png)

### Step 4

Start your server and make sure it says "Online".

![](https://i.imgur.com/LIGfDmI.png)

### Step 5

Open Steam & click on the **View** button on the top left. In the dropdown menu, click **Game Servers**.

![](https://i.imgur.com/JPIiB52.png)

### Step 6

Go to Favorites and click on the <Code>+</Code> icon.

![](https://i.imgur.com/9XVBvH8.png)

### Step 7

Paste the IP of your server next to the new port you created in step 2 and 3.

![](https://i.imgur.com/ChUr8Fj.png)


### Step 8

Finally select the server and click on **Connect**.

![](https://i.imgur.com/75kH8Qi.png)

</Steps>
